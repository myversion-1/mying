# 客户 Logo 图片说明

## 图片规格要求

### 尺寸要求
- **推荐尺寸**：400x200px（2:1 宽高比）
- **最小尺寸**：300x150px
- **最大尺寸**：800x400px
- **格式**：PNG（支持透明背景）或 SVG（矢量图，推荐）

### 设计规范
- **背景**：透明背景（PNG）或白色背景
- **颜色**：单色或双色 Logo，适合深色背景显示
- **风格**：简洁、专业、现代
- **文字**：清晰可读，避免过小字体

### 文件命名
请使用以下文件名（对应 `src/content/partners.ts` 中的配置）：

1. `theme-park-inc.png` - Theme Park Inc. (美国主题公园)
2. `dubai-entertainment.png` - Dubai Entertainment Group (阿联酋娱乐集团)
3. `shanghai-mall.png` - Shanghai Mall Group (中国购物中心集团)
4. `family-fun-centers.png` - Family Fun Centers (美国家庭娱乐中心)
5. `tokyo-amusement.png` - Tokyo Amusement Co. (日本娱乐公司)
